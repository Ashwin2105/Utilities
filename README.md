# Utilities
A collection of python scripts that may come in handy in everyday life at IIT Mandi

# Install

```
pip3 install requests
pip3 install bs4
pip3 install lxml
pip3 install pandas
pip3 install getpass
pip3 install threading
```

# Run
## Linux
```
python3 [scriptname.py]
eg :- python3 SGPA.py
```
